# Phase II: Full-Stack Todo Application

## Tech Stack Requirements

1.  **Frontend:** Next.js 16+ (App Router), TypeScript, Tailwind CSS, Shadcn UI (for modern, eye-catching look).
2.  **Backend:** Python FastAPI, SQLModel (ORM), Pydantic V2.
3.  **Database:** Neon Serverless PostgreSQL.
4.  **Auth:** Better Auth (Client-side) with JWT Plugin. Backend must verify JWT tokens.
5.  **Architecture:** Monorepo. Strict separation between Frontend and Backend. RESTful API.

## Design Principles

-   **UI:** Must be modern, minimalistic, clean, and responsive (Mobile/Desktop).
-   **Security:** No unauthenticated access to APIs. User data isolation (User A cannot see User B's tasks).
-   **Code:** No "jugaar". Professional, modular code. Type-safe implementation.
